## 一、导数

函数 `f(x)` 在某点的导数：

```
f'(x) = lim(h→0) [f(x+h) - f(x)] / h
```

**含义**：x 增大一个极小量 Δx，f(x) 大约增大 `f'(x) * Δx`。几何上是曲线在该点的切线斜率。

## 二、偏导数

函数有多个输入时，固定其他变量不动，只对一个变量求导。

```
f(x, y) = x^2 + 3xy + y^2

∂f/∂x = 2x + 3y   （把 y 当常数，对 x 求导）
∂f/∂y = 3x + 2y   （把 x 当常数，对 y 求导）
```

**含义**：`∂f/∂x = k` 表示固定 y 不动，x 增大 1，f 大约增大 k。

在点 `(x=1, y=2)` 处：

```
∂f/∂x = 2*1 + 3*2 = 8   （x 增大 1，f 大约增大 8）
∂f/∂y = 3*1 + 2*2 = 7   （y 增大 1，f 大约增大 7）
```

## 三、梯度

把所有偏导数收集成一个向量：

```
∇f(x, y) = [∂f/∂x, ∂f/∂y]
```

**含义**：梯度向量指向函数值增长最快的方向，梯度的反方向是下降最快的方向。

**例子**：`f(x, y) = x^2 + 3xy + y^2`，在点 `(1, 2)` 处：

```
∂f/∂x = 2*1 + 3*2 = 8
∂f/∂y = 3*1 + 2*2 = 7

∇f(1, 2) = [8, 7]
```

沿 `[8, 7]` 方向走，f 增长最快；沿 `[-8, -7]` 方向走，f 下降最快。

## 四、链式法则

### 单变量

`y = f(u)`，`u = g(x)`，则：

```
dy/dx = (dy/du) * (du/dx)
```

直觉：x 变化先被 g 放大，再被 f 放大，总放大倍数是两段相乘。

**例子**：`y = (x^2 + 1)^3`

```
令 u = x^2 + 1，则 y = u^3

du/dx = 2x
dy/du = 3u^2

dy/dx = 3u^2 * 2x = 3(x^2+1)^2 * 2x = 6x(x^2+1)^2
```

**三层复合**：`y = f(g(h(x)))`

```
dy/dx = (dy/dg) * (dg/dh) * (dh/dx)
```

链条有多长，就乘多少项。

### 多变量（重要）

`L = f(u, v)`，u 和 v 都依赖 x：

```
dL/dx = (∂L/∂u)(du/dx) + (∂L/∂v)(dv/dx)
```

**规律**：x 通过几条路径影响 L，就有几项，全部加起来。

**例子**：`z = x^2 + y^2`，`x = t`，`y = 2t`

```
∂z/∂x = 2x，dx/dt = 1
∂z/∂y = 2y，dy/dt = 2

dz/dt = 2x * 1 + 2y * 2 = 2t + 8t = 10t

验证：z = t^2 + 4t^2 = 5t^2，dz/dt = 10t  ✓
```

## 五、反向传播

### 本质

反向传播 = 链式法则在复合函数上的系统化执行。

### 符号

定义 `δv = ∂L/∂v`，表示 L 对中间变量 v 的偏导数，称为"v 的梯度"。

### 核心规则

对于计算中的一步 `v = f(u)`：

```
δu += δv * (∂v/∂u)
```

用 `+=` 是因为 u 可能有多条出路，每条路径传回来的梯度要累加。

### 算法

```
第一步（前向）：从输入到输出，计算每个中间变量的值，全部存下来

第二步（反向）：
  初始化 δL = 1
  按前向的逆序，对每个中间变量 v：
    对 v 的每个输入 u：
      δu += δv * (∂v/∂u)

结果：每个输入变量的 δ 就是 L 对它的偏导数
```

### 常见操作的局部导数

```
v = u + c    →  ∂v/∂u = 1        （加常数，梯度原样传过去）
v = u * c    →  ∂v/∂u = c        （乘常数，梯度乘以该常数）
v = u * w    →  ∂v/∂u = w        （乘法，梯度乘以另一个输入的值）
v = u^n      →  ∂v/∂u = n*u^(n-1)
v = e^u      →  ∂v/∂u = e^u
v = ln(u)    →  ∂v/∂u = 1/u
v = sigmoid(u) = 1/(1+e^(-u))  →  ∂v/∂u = v*(1-v)
```

### 例子一：线性链（无分叉）

`x=2`，`u = x^3`，`v = u + 5`，`L = v^2`

前向：

```
u = 8，v = 13，L = 169
```

反向：

```
δL = 1
δv = δL * 2v = 1 * 26 = 26
δu = δv * 1  = 26 * 1 = 26   （v = u+5，∂v/∂u = 1）
δx = δu * 3x^2 = 26 * 12 = 312
```

验证：`L = (x^3+5)^2`，`dL/dx = 2(x^3+5)*3x^2 = 2*13*12 = 312`  ✓

### 例子二：有分叉（变量出现在多个位置）

`L = (a + b) * (b + 1)`，`a=2, b=3`

拆开：

```
c = a + b = 5
d = b + 1 = 4
L = c * d = 20
```

反向：

```
δL = 1

δc = δL * (∂L/∂c) = 1 * d = 4
δd = δL * (∂L/∂d) = 1 * c = 5

δa = δc * (∂c/∂a) = 4 * 1 = 4

δb = δc * (∂c/∂b) + δd * (∂d/∂b)   ← b 有两条出路，累加
   = 4 * 1 + 5 * 1 = 9
```

验证：`L = ab + a + b^2 + b`，`∂L/∂a = b+1 = 4`，`∂L/∂b = a+2b+1 = 9`。✓

### 关键结论

单变量出现在多个位置时，反向传播的累加规则和多变量链式法则是同一件事：

```
x 通过路径 u1, u2, ... 影响 L：

δx = δu1*(∂u1/∂x) + δu2*(∂u2/∂x) + ...
```

反向传播没有发明新东西，只是把链式法则在计算图上高效地执行了一遍。
